# Student Knowledge Visualization App

## Overview
This is a basic front-end for a two-column application designed to:
- Visualize students' knowledge and interaction from annotation activities (Left Column)
- Provide a collaborative writing space (Right Column)

## Features
- Responsive two-column layout
- Local storage for collaborative writing space
- Placeholder for knowledge visualization

## Setup
Simply open `index.html` in a modern web browser to use the application.
